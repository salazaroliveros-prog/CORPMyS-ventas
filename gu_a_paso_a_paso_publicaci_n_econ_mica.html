# Guía de Publicación Económica: Plataforma ArchiCraft & Heirloom Crust Para publicar tu plataforma de arquitectura sin gastar mucho dinero pero manteniendo la profesionalidad, te recomiendo este "paso a paso" utilizando herramientas de vanguardia y bajo costo. ## 1. Dominio (.com o .com.gt) * **Donde comprar:** [Namecheap](https://www.namecheap.com/) o [Cloudflare](https://www.cloudflare.com/products/registrar/). * **Costo:** Aproximadamente $10 - $15 USD al año (Q80 - Q120). * **Tip:** Evita comprar dominios directamente en el hosting para que siempre tengas el control total. ## 2. Hosting (Alojamiento) de Bajo Costo y Alta Velocidad Como tu plataforma está enfocada en diseños técnicos y archivos pesados (DWG/PDF), necesitas algo rápido: * **Vercel o Netlify (Recomendado):** Tienen planes **GRATUITOS** para proyectos personales y de bajo tráfico. Son ideales para sitios rápidos y modernos. * **Hostinger:** Si prefieres un panel tradicional, su plan sencillo cuesta unos $2.99 USD al mes (Q25). ## 3. Base de Datos y Almacenamiento de Planos (Archivos Pesados) * **Supabase:** Ofrece una base de datos PostgreSQL y almacenamiento de archivos con un nivel gratuito muy generoso. Aquí guardarías los datos de ventas y los archivos PDF de los planos. * **Cloudinary:** Ideal para las fotos de los proyectos arquitectónicos, optimizando el peso para que la web cargue rápido. ## 4. Pasarela de Pagos (PayPal y Bancos) * **PayPal:** No tiene costo fijo mensual, solo cobran una comisión por cada venta exitosa (aprox. 5.4% + $0.30). * **Pagos con Tarjeta (Guatemala):** Puedes usar [Pagalo](https://www.pagalo.gt/) o [NeoNet](https://www.neonet.com.gt/). Tienen procesos de integración profesional para Quetzales. ## 5. Implementación de la IA Gemini * **Google Cloud / Firebase:** Puedes usar las funciones de Firebase para conectar la API de Gemini. Google ofrece créditos gratuitos iniciales (Cloud Free Tier) que suelen cubrir el inicio de un proyecto pequeño. ## Paso a Paso para la Publicación: 1. **Exportar el Código:** Haz clic en el botón **"</> View Code"** aquí en Stitch para descargar el HTML/CSS. 2. **Subir a GitHub:** Crea un repositorio gratuito en GitHub y sube tu código. 3. **Conectar con Vercel:** Conecta tu cuenta de GitHub a Vercel. Se publicará automáticamente cada vez que hagas un cambio. 4. **Configurar el Dominio:** En el panel de Vercel, añade tu dominio (ej. www.estudioartesanal.com.gt). 5. **Activar SSL:** Vercel te da el certificado de seguridad (el candadito) **GRATIS**. --- *Documento preparado por tu asistente de diseño Stitch para Heirloom Crust Architectural Services.*
