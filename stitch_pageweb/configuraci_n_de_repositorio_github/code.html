Configuración del Repositorio

Establece la base técnica de tu proyecto. Sigue estos pasos para vincular tu arquitectura Heirloom Crust con el mundo exterior.

cloud_upload
1

Crear Repositorio en GitHub

Inicia sesión en tu cuenta de GitHub y crea un nuevo contenedor para el código.

2

Inicializar Código Localmente

Ejecuta estos comandos en la terminal de tu proyecto para preparar el seguimiento de cambios.

bash
git init
git add .
git commit -m "initial: primer amasado del código"
3

Conectar Remoto y Subir

Paso A: Vincular origen

git remote add origin https://github.com/usuario/heirloom-crust.git

Paso B: Realizar Push

git push -u origin main
