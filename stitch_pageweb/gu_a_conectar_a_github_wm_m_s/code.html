Manual de Estructura Digital

Despliegue de Proyecto en GitHub

Protocolo técnico para la vinculación de activos digitales. Siga los planos secuenciales para establecer una conexión sólida entre su entorno local y la infraestructura de nube.

05
Pasos Estructurales
architecture

01. Cimentación del Repositorio

Inicie sesión en su consola de GitHub y cree un nuevo espacio de almacenamiento. Defina el nombre del proyecto con precisión técnica, evitando espacios y caracteres especiales.

Acción Requerida

Botón "New" > Ingrese "Repository name" > "Create repository"

construction

02. Inicialización Local

Prepare su carpeta local para el control de versiones. Este comando establece el "blueprint" invisible de Git.

git init
inventory_2

03. Registro de Materiales

Añada todos los archivos al área de preparación (staging) y consolide los cambios con un mensaje descriptivo que identifique esta versión inicial.

A.
Stage
B.
Commit
git add .
git commit -m "Estructura inicial del proyecto"
payments

04. Enlace Estructural

Vincule su infraestructura local con el nodo remoto en la nube de GitHub utilizando la URL proporcionada durante la creación del repositorio.

git remote add origin https://github.com/usuario/proyecto.git
dashboard

05. Carga Final

Transfiera sus archivos a la rama principal. Una vez completado, el proyecto estará disponible en el entorno global de GitHub.

git push -u origin main
